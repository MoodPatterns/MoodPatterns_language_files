Gráficos de Cursos

Os gráficos mostram o passado recente de suas escalas selecionadas. Para ajudá-lo a detectar tendências no zigue-zague de altos e baixos, você pode escolher exibir as curvas suavizadas (esta opção só fica disponível quando há dados suficientes para suavização).

Exceto para escalas personalizadas (para as quais você pode escolher livremente os pontos finais), os valores mais altos são normalmente melhores do que os valores mais baixos. O eixo na parte superior do gráfico indica a hora e a data. Você pode pinçar e ampliar o gráfico para obter datas mais precisas.

Se os gráficos demorarem muito para carregar, você pode acelerar o processo reduzindo o número de dias exibidos. Para fazer isso, vá para Configurações ( ⋮ ) -> Funcionamento -> Tamanho retrospectivo.

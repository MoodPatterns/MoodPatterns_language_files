Alerts

You can set alerts to monitor your mood and be notified if something changes.

To ensure meaningful notifications, there must be at least one entry per day on average so that an alarm can be triggered.

Alert settings

To set an alert you need to decide on a scale and the type, the direction, the size, and the duration of the change you want to look out for.

Scale

Select the scale of interest.
Note: You can choose all active scales. Pro scales will be monitored, even without Pro status. However, without Pro, the display of the triggering sequence is only available once, by clicking the alert notification.

Type

Describes the kind of change that should be monitored.

  • LEVEL: general increase or decrease; monitors: average value of the scale
  • VARIATION: changes in the variation; monitors: variance of the scale
  • DYNAMICS: changes in how similar adjacent entries are; monitors: mean absolute distance between successive entries

Direction

Describes the direction of change that should be monitored.

  • INCREASE: you will be alerted if the monitored characteristic increases
  • DECREASE: you will be alerted if the monitored characteristic decreases

Size

Sets the required minimum change to trigger an alert.

LEVEL
  • SMALL: 0.2 standard deviations
  • MEDIUM: 0.5 standard deviations
  • LARGE: 0.8 standard deviations
VARIATION & DYNAMICS
  • SMALL: 20 %
  • MEDIUM: 50 %
  • LARGE: 80 %

Note: That means that the size is relative to the usual variation of the scale values. If regularly there is not much change, then even a small value drift will trigger an alert. However, if big jumps are commonplace, then a change needs to exceed those to get noticed.

Duration

The duration sets the time-frame of the observational period and its preceding control period.

Alert management

You can have an arbitrary number of alerts.

If you want to stop an alert you have three options:

  1. You can deactivate it with the switch in its top-left corner.
  2. You can archive it. This moves the alert to the ARCHIVE, from where you still have access to its data and can restore it anytime you like.
  3. You can delete it. This will permanently remove the alert and all data associated with it.

If you edit an existing alert, then a new alert will automatically be saved, and the previous version will be archived.
